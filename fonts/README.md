# Fonts

`Fraunces-latin.woff2` and `Fraunces-Italic-latin.woff2` are the variable
Fraunces family (opsz + wght axes), latin subset, self-hosted so the app makes
no third-party request and works under the project-pages base path.

Fraunces is licensed under the SIL Open Font License 1.1.
Source: https://github.com/undercasetype/Fraunces
The woff2 subsets were produced by the Google Fonts serving pipeline.
